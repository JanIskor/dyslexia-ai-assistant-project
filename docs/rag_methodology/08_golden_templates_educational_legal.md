# Golden adaptation templates for educational and legal genres

## Назначение документа

Документ задаёт эталонные шаблоны результата для наиболее чувствительных и приоритетных жанров адаптации.

Шаблоны нужны:
- для стабилизации output структуры;
- для уменьшения случайного rewriting;
- для повышения предсказуемости адаптации;
- для повышения читаемости материалов для обучающихся с дислексией.

---

# Educational genre

## Общий принцип

Учебный текст допускает:
- controlled rewriting;
- синтаксическое упрощение;
- пояснение терминов;
- структурную перестройку.

Но:
- предметный смысл;
- термины;
- причинно-следственные связи;
- научная точность

обязаны сохраняться.

# [Тема]

Краткое вводное объяснение.

## 1. Что это такое?

Описание процесса или механизма.

## 3. От чего зависит?

Факторы, условия, ограничения.

## 4. Почему это важно?

Практическое значение или итог.

## Вывод

Краткое закрепление главной мысли.

Основные правила

* одно смысловое действие = один блок;
* термины сохраняются;
* новый термин можно кратко пояснить;
* нельзя вводить новые научные факты;
* нельзя заменять термин бытовым аналогом без сохранения термина.
